# 快速上手

本页帮助您快速上手 ISLANDS 小游戏

相关部分：[乐魂战](#乐魂战)｜高炉对垒

## 乐魂战

乐魂战是一个 Minecraft 大型多人团队对抗游戏，对局时长在30-40分钟左右。

游戏开始后玩家会随机分队。每队伍会根据人数生成1只及以上乐魂。

对局中，玩家主要目标是守护自己队伍的乐魂，摧毁其他队伍的乐魂，最后击败其他队伍的玩家。

当玩家死亡时，如果场上还有其队伍乐魂存活，玩家可以复活，如果没有该队伍乐魂存活，则玩家无法复活。

当场上只有最后一支队伍存活或其乐魂长时间未被摧毁时，游戏结束，该队伍胜利。

### 准备阶段

在此阶段，玩家需要从54个随机配件中选择5个配件保留下来。

配件可以安装至其限制的工具与武器，并且给予加成。

配件分为五个不同的品质等级，等级越高加成越强。

品质等级：白色 < 绿色 < 蓝色 < 紫色 < 金色

这54个随机配件会从低品质等级排序到高品质等级。

如果玩家没有选择/选满5个配件，系统会自动补全随机配件中最高品质的配件

一个装备最多安装三个配件，同类型配件不可叠加给一个装备。

玩家会被给予石剑，石斧，石稿，木铲，望远镜，初始弹射器，收纳袋，以及初始资源。

### 发育阶段

发育阶段内靠近乐魂的玩家将会有急迫，速度，抗性提升，生命恢复加成。靠近其他队伍的乐魂不会受到加成。

乐魂上绑定着一个熔炉。熔炉是玩家升级自身以及乐魂装备的唯一途径。右键熔炉可以进入核心商店。

在核心商店中，玩家可以消耗背包中的资源升级或购买道具。每个队伍都拥有团队箱子，可以通过核心熔炉访问。

当背包中资源不充足时，玩家升级或购买装备时会消耗团队箱子内的资源。

游戏一共有五种基础资源：

原木 :log: - 最常见、最易采集、 用量最大。

铁 :iron\_ingot: - 常见、用途广泛、基础资源、用于各种升级。

铜 :copper\_ingot: - 少见、用量少、用于[冶炼](#冶炼)和部分装备升级。

煤炭 :coal: - 较常见、易采集、用途较少、用于核心熔炉升级。

雪块 :snow\_block: - 分布区域极少、极易采集、用于乐魂升级和回血。

以及两种高级资源：

紫水晶碎片 :amethyst\_shard: - 紫晶矿区独有。稀有、分布集中、生长缓慢、用于装备高阶升级。

下界合金 :netherite\_ingot: - 通过铁和铜[冶炼](#冶炼)获得、可用于装备升级。

这些资源（除下界合金）会分布在不同的岛上，挖掘这些资源会直接进入背包。资源被挖掘后会随着时间再生。

游戏开始时玩家所在的初始岛拥有除紫水晶和雪块之外的所有资源。

当玩家使用初始稿子挖掘铁以及铜矿石时，玩家会获得粗矿石。玩家需要手持粗矿石，对着核心熔炉长按右键进行烧制。如果玩家将稿子升到三阶后（合金稿），稿子将会拥有熔炼效果，此时挖掘原矿将会直接获得烧制后的矿石。

需要注意的是，玩家升级装备时可升的最大等级受到熔炉的限制。如果需要升级稿子到合金稿需要先升级核心熔炉至核心高炉。升级核心熔炉会增加熔炉最大炉温。炉温需要玩家手持煤炭对着核心熔炉使用补充。炉温会影响到乐魂飞行速度以及[冶炼](#冶炼)速度。

原木可以通过挖掘地图上任意种类的木头获得（挖掘木板不会获得原木）。放置原木时，原木将会变成木板，并且向原木放置朝向延伸出四格原木，合计5格木板。

在地图“紫晶矿区”中，雪块只能在中心岛采集。挖掘雪块会获得雪球，雪球会自动合成为雪块。
